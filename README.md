# Poverty Estimation from Satellite Imagery

This project uses geospatial AI to predict poverty levels from satellite images.

## Folder Structure

- `data/` — Datasets (raw and processed)
- `notebooks/` — Jupyter notebooks for exploration and prototyping
- `models/` — Saved models and model code
- `preprocessing/` — Data cleaning and preparation scripts
- `training/` — Model training and evaluation scripts
- `visualization/` — Scripts for visualizing data and results
- `configs/` — Configuration files for experiments

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Explore the example notebooks in `notebooks/`.
3. Add your data to the `data/` folder.

## Requirements
See `requirements.txt` for a list of required Python packages.
